Project Title: ESP32-CAM Telegram Security System
What it does: This project turns an ESP32-CAM module into a smart security camera. When the system detects movement using OpenCV, it automatically:

Sends a snapshot to your Telegram.
Triggers a physical alarm (buzzer).

Key Features: 
- Low-latency motion detection via Python/OpenCV.
- Real-time notifications via Telegram Bot API.
- Physical alarm system integration.
- Automated camera connection recovery.
